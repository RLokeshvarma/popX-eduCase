# PopX-EduCase

A pixel-perfect implementation of the PopX mobile UI using **React + TypeScript + Vite** based on the provided Adobe XD design.

---

## Live Demo

Vercel Deployment  
[View live](https://popxeducase.vercel.app/)

---

## Tech Stack

- **React** – Component-based UI development
- **TypeScript** – Type safety and better code maintainability
- **Vite** – Fast development server and optimized build
- **CSS** – Custom styling to match the Adobe XD design

---

## React Libraries Used

### React Router DOM
Used for handling navigation between application pages.

Pages implemented using routing:
- Landing Page
- Login Page
- Signup Page
- Profile Page

### React Icons
Used to display the **camera icon on the profile avatar**.

---

## Features

- Pixel-perfect UI based on Adobe XD design
- Mobile app style layout centered on desktop
- Full screen layout on mobile devices
- Floating label form inputs
- Required form validation with red asterisk
- Fixed action buttons
- Profile screen with avatar and camera icon
- Scrollable mobile container layout
